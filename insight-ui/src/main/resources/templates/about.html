实时监控异常请求，智能识别错误模式和高风险操作

性能分析

深度性能分析，提供QPS、响应时间、错误率等关键指标

技术架构

  • 数据采集层： 基于Spring AOP和Servlet Filter，无侵入式采集链路数据
  • 数据传输层： 异步上报机制，支持批量处理和断点续传
  • 数据处理层： 实时数据清洗、聚合和分析，支持多种存储后端
  • 数据展示层： 基于ECharts的可视化界面，支持实时刷新和交互式查询
  • 存储层： 支持H2（嵌入式）、MySQL、PostgreSQL等多种数据库

快速开始

只需三步，即可开始监控您的Spring Boot应用

1
添加依赖
<dependency>
    <groupId>io.github.iweidujiang</groupId>
    <artifactId>spring-insight-core-spring-boot-starter</artifactId>
    <version>0.1.0-SNAPSHOT</version>
</dependency>
2
启用注解
@SpringBootApplication
@EnableSpringInsight
public class MyApplication {
    public static void main(String[] args) {
        SpringApplication.run(MyApplication.class, args);
    }
}
3
访问监控

启动应用，访问：

http://localhost:8081/insight-ui
